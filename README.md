# SDL-Nearest-Neighbour
Nearest Neighbour Visualizer using SDL

## Build from source

```bash
git clone https://github.com/valdaarhun/SDL-Nearest-Neighbour.git
cd SDL-Nearest-Neighbour
make
```

## Execution

```
./nn_viz
```

## Usage

- Left click on squares to drag them around.
- Press 'Q' to quit.
